# Shopping Cart (React + TypeScript)

## Опис: Показан функціонал магазину меблів з кошиком, збереженням даних у LocalStorage та інтеграцією з Firebase Firestore.

# Технологічний стек:

## React (Functional Components, Hooks)
## Redux Toolkit (State management)
## TypeScript (Interface definitions, Type safety)
## Firebase (Database for orders)
## CSS Modules / CSS (Styling)

# Key Features (Головні фішки):

## Повна типізація даних за допомогою TypeScript.
## Persistence: кошик не зникає після оновлення сторінки (LocalStorage).
## Відправка замовлень у реальному часі до Firebase.
## Адаптивна верстка SideBar для перегляду обраних товарів.

# Установка:
1. `clone repository`
2. `npm install`
3. `npm run start`
